Baybayin Info

What is Baybayin?

Baybayin is an ancient pre-colonial script used in the Philippines. It is a beautiful and unique writing system that was used to write Tagalog, Ilocano, Pangasinan, and other Philippine languages before the arrival of the Spanish.

 The Baybayin Alphabet

Baybayin consists of 17 basic characters: 3 vowels and 14 consonants. Each consonant has an inherent 'a' sound, which can be changed with diacritical marks called kudlit.

 Basic Writing Rules
  • Each Baybayin character represents a syllable (consonant + vowel).
  • To change the vowel sound, add a kudlit above (for 'e'/'i') or below (for 'o'/'u') the character.
  • To write a pure consonant (no vowel), use a special mark called virama (not used in all historical texts).
  • There are no separate characters for ending consonants; context is used.
 Stroke Order

Baybayin is written from left to right. Each character is written in a few simple strokes. Practicing the correct stroke order helps with writing speed and style.

 History

Baybayin was widely used before Spanish colonization. The script declined as the Latin alphabet became dominant, but it remains a symbol of Filipino identity and heritage. Today, Baybayin is experiencing a revival among artists, educators, and cultural advocates.
